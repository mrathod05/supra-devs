## Installation

### Prerequisites

- Node.js 16+
- React 19.2.1+
- Starkey Wallet Extension (for browser integration)

### NPM

```bash
npm install @mrathod05/supra-devs
```

### Yarn

```bash
yarn add @mrathod05/supra-devs
```

### PNPM

```bash
pnpm add @mrathod05/supra-devs
```
